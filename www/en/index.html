What's this?

Kijimuna is an Eclipse plugin that supports application development using Seasar. It provides following features.

  • Dicon Editor
    • Syntax Highlighting
    • Content Assistant for XML-tags, class name, OGNL and more.
    • Hyperlink by CTRL+Click
    • DTD based validation
  • Outline
    • Display of a component tree
    • Display of a reference of component
    • Hyperlink by double click
  • Validation
    • Dicon file attribute and OGNL check

Setup

Kijimuna works with Eclipse 3.1. You can install it from update site.

  • http://eclipse.seasar.org/updates/3.1/

Development for Eclipse3.0 is finished. You can install an old version from update site.

  • http://eclipse.seasar.org/updates/3.0/

Usage

You have to configure your project properties.

  • Choose [Kijimuna] in the your project properties, and check [Seasar2 Project].
